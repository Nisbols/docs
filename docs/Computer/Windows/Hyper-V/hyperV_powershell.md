# Hyper-V Powershell

## Commands
  ```ps
 Get-VM
 ```
 * Displays list of VM

```ps
Start-VM
```
* Starts a VM

```ps
Stop-VM
```
* Shuts down a VM
  * Check the Docs below for options of different ways to shutdown a VM
